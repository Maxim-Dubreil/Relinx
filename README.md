# Relinx

Relinx est un jeu de mots casual où tu dois trouver le mot caché qui relie deux mots donnés.

> Exemple : DENT + OR → COURONNE

La première version du jeu se concentre sur un **mode logique quotidien** : une énigme partagée par jour, avec un nombre d’essais illimité et un retour sur la proximité de chaque proposition.

---

## Tech stack

- **Framework** : Next.js (React)
- **Langage** : TypeScript
- **Runtime** : Node.js
- **Cible** : Web (desktop & mobile), déploiement sur Vercel

## Démarrage du projet

### Pré-requis

- Node.js (version LTS recommandée)
- Un gestionnaire de paquets : pnpm

### Installer les dépendances

```bash
pnpm install
```

### Lancer le serveur

```bash
pnpm run dev
```
